ITO YUNG STEP BY STEP KUNG PAANO I-SET-UP GALING GITHUB PAPUNTA SA VS-CODE KAYO ANG (CLONE) 

download/CLONE nyo muna Pag-click nyo ng "Open with GitHub Desktop" sa  GitHub repository page, kusa nang magbubukas ang GitHub Desktop app sa laptop nyo.

gagawa kayo ng folder lagay nyo sa pangalan ay barangay_system wag na yung mahaba tulad nito Barangay Appointment and Queue Management System ang haba kase: 
pag tapos Tatanungin kayo ng GitHub Desktop kung saan nila gustong i-save ang folder ng iyong system.

tapos open nyo yung sa VS Code: Kapag na-clone na nyo na ang repository, may makikita kayong button sa loob ng GitHub Desktop na may label na "Open in Visual Studio Code".

Pag-click nyo doon, kusa nang magbubukas ang VS Code at doon na mismo sa folder ng iyong Barangay System.
